# My First API Blog Post

This is a sample blog post created using the QuickBlog API client!

## Features

- **Easy to use** - Simple CLI commands
- **Markdown support** - Write in your favorite format
- **Fast publishing** - Instant deployment to your blog

## Getting Started

1. Install the CLI
2. Login with your credentials
3. Start blogging!

This post was created using:
```bash
quickblog create --file sample-post.md
```
